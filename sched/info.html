ABOUT

This course educates students about managing information within an organization with an emphasis on developing a system to manage data/information. It aims to orient students on the different IT applications used in handling information within an organization. Also, it strives to teach them how to create a simple system for managing data/information. Topics include information management, database design, data modeling, SQL, and implementation using C# for front-end and MS SQL Server for the back-end. Students would have to defend the database they designed and the application software that they developed.
